# 03_Regional_Ingredients

**Anno 1800** regional ingredient mappings (e.g. Africa DLC replacements).

| File | Purpose |
|------|---------|
| `regional_ingredients.json` | Maps base ingredients to regional variants during processing |

## When it applies

- Anno 1800 extractions that use regional content rules in the pipeline.
- Not used for Anno 117-only runs.

## Editing

- JSON object — keep valid UTF-8.
- Back up before large edits; invalid mappings can produce wrong asset references.

Structure and field meanings: [config README](../README.md) (Regional ingredients section).
